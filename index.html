Configuração do EmailJS

Dados da sua conta em emailjs.com — usados para disparar as cobranças semanais dos checklists
Salvo ✓
No template do EmailJS, use as variáveis {{to_email}}, {{to_name}}, {{setor}}, {{tipo_checklist}} e {{link_formulario}}. Como os dados deste app são compartilhados, qualquer pessoa com acesso ao link também acessa essas chaves salvas aqui — prefira preencher a Private Key só se o envio pela Public Key estiver sendo bloqueado por checagem de origem.
